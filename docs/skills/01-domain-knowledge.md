# 입찰 도메인 지식
> 한국 공공조달 입찰(전기/통신/소방)의 핵심 메커니즘
> 복수예비가격, 사정률, 낙찰하한율, 적격심사 규칙

## 핵심 용어

| 용어 | 정의 |
|---|---|
| **기초금액** | 복수예비가격 작성의 기준 금액. 추정가격 + 부가세 (조달청 예외 있음). 조달청은 입찰일 7일 전 발표 |
| **추정가격** | 적격심사 대상의 기준이 되는 가격. 부가세 미포함 |
| **공사추정금액** | 추정가격 + 부가세 + 관급액. 조달청 시공비율 산정 기준 |
| **추정금액** | 추정가격 + 관급액 |
| **관급자재비** | 발주처가 제공하는 자재비. 도급자설치/관급자설치로 구분 |
| **A값** | 도급자가 받지만 관급으로 처리되는 금액. 투찰금액 계산식에 영향 |
| **복수예비가격** | 기초금액 ±2~3% 범위에서 만든 15개 비공개 예비가격. 4개 추첨 후 산술평균 = 예정가격 |
| **사정률** | (예정가격/기초금액 - 1) × 100. 기초금액 대비 예정가격의 증감률 |
| **예정가격** | 15개 복수예비가격 중 4개 추첨된 평균 |
| **낙찰하한율** | 적격심사 통과를 위한 최저 투찰율. 기관·금액·시기별 다름 |

## 사정률 표기 규칙 (이 시스템 전용)

이 시스템은 **반드시 100% 기준**으로 사정률을 표기합니다:

```
0-base (내부 계산): -0.22%
100-base (UI 표시): 99.78%

변환 공식: 100-base = 100 + 0-base
```

DB의 `br1` 컬럼은 100-base로 저장됩니다. 0-base가 필요하면 `br1 - 100`. 사용자에게 보이는 모든 화면·문서·AI 응답에서 100% 표기를 사용합니다.

## 복수예비가격 추첨 메커니즘

```
1. 발주처가 비공개로 15개 예비가격 작성 (기초금액 ±3% 또는 ±2%)
2. 입찰 참여 업체들이 각각 2개 번호씩 선택 (블라인드)
3. 가장 많이 선택된 4개 번호가 추첨됨
4. 동률일 경우 빠른 번호 자동 선택
5. 4개 예비가격의 산술평균 = 예정가격
6. 사정률 = (예정가격 / 기초금액 - 1) × 100
```

**구조적 특성**:
- C(15,4) = 1,365가지 조합
- 같은 기관에서도 사정률 std ≈ 0.6~1.0%
- **이론적 예측 한계**: 0.642% MAE (51K건 측정)
- 참여자 수, 금액대, 업종으로는 사정률 예측 거의 불가능

## 투찰금액 산출식

```
A값 있을 때:
  투찰금액 = A값 + (예정가격 - A값) × 낙찰하한율

A값 없을 때:
  투찰금액 = 예정가격 × 낙찰하한율

소수점 이하: 절상 처리
LH만 특수: 예정가격 천원 이하 절상
```

## 적격심사 점수 체계 (전기/통신/소방)

종합평점 95점 이상이 낙찰자.

### 점수 구성 (3억 미만 기준)
```
시공경험평가: 5점 (5년 실적 ≥ 기초금액 × 0.5 시 만점)
경영상태평가: 15점
  - 부채비율: 5점 (자사부채비율 < 평균부채비율 × 100%)
  - 유동비율: 5점 (자사유동비율 ≥ 평균유동비율 × 100%)
  - 영업기간: 1점 (3년 이상 시 만점)
  - 신인도: 4점
입찰가격평가: 90점
  산식: 90 - 20 × |88/100 - 입찰가격/예정가격| × 100 = 85점
  → 만점 조건: 입찰가격/예정가격 = 87.745%
```

### 점수 부족 시 보완
경영상태나 시공경험에서 1점 부족할 때:
```
입찰가격으로 86점을 받으면 됨
산식 역산: 86 = 90 - 20 × |0.88 - x| × 100
→ x = 0.87795 → 87.795% (낙찰하한율보다 0.05%p 높게 투찰)
```

## 1순위 낙찰 메커니즘

```
1. 모든 참여자가 투찰 (기초금액 기반)
2. 개찰: 예정가격 발표
3. 예정가격 이하 + 낙찰하한율 이상의 투찰자 중
   가장 낮은 가격 = 1순위
4. 1순위 업체의 적격심사 (95점 이상이면 낙찰)
5. 부적격 시 2순위로 이전
```

**1순위 업체의 투찰 패턴 (162건 분석)**:
- 낙찰하한율 대비 마진: 중앙값 +0.001~0.005% (소수점 4자리 경쟁)
- "가정 사정률"을 매우 정밀하게 계산해서 투찰
- 결국 사정률 예측 정확도보다 **소수점 상한율 정밀도가 승부수**

## 공동수급협정 (참고)

### 공동이행
같은 업종 2개 이상 업체가 시공비율 분할
- 예: 전기 A 51% + 전기 B 49%

### 분담이행
다른 업종 업체가 면허 보완을 위해 분담
- 예: 전기 A 100% + 소방 B 100%

### 의무 공동도급
지역업체와 반드시 공동 (비율 30~49% 이상)

### 일반 공동도급
선택적 (지역제한 없을 때)

## 기관유형 분류 (`clsAg` 함수)

```
/조달청/      → "조달청"
/교육/        → "교육청"
/한국전력|한전/ → "한전"
/LH|주택공사|토지주택/ → "LH"
/군|사단|국방|해군|공군|육군|해병/ → "군시설"
/수자원/      → "수자원공사"

```

**주의**: 서울교통공사·한국철도공사·한국가스공사 등은 "지자체"로 분류되지만 실제 운영 패턴은 매우 다름. Phase 12-C 발주사별 모델이 이 차이를 잡아냄.

## 실전 인사이트 (51K건 분석 결과)

### 사정률 예측에 효과 없는 것
- **참여업체수**: R² 거의 0. 복수예가가 랜덤이라 무관
- **금액대**: 중앙값 변동 0.05%p 이내. 비율 기준이라 절대금액 영향 적음
- **업종(전기/통신/소방)**: R² -71.9%. 사실상 동일

### 사정률 예측에 효과 있는 것
- **발주사 단독**: R² 28.8% (가장 강력한 단일 지표)
- **발주사 × 금액대**: R² 22.7% (과적합으로 약간 감소)
- **발주사별 오프셋 보정**: MAE 7% 개선 (Phase 12-D 검증)

### 낙찰 가능 마진 (P1~P5별)
- **P1**: P75 마진 0.00% (상위 25%에서 낙찰 가능)
- **P5**: 전 구간 -2.14% (구조적 낙찰 불가)
- **마진 -2%**: 법정 하한과 1위가 같음 = 낙찰 불가능

### 가정 사정률 인사이트
1순위 업체는 낙찰하한율 + 0.001~0.005% 마진 투찰. 즉:
- 사정률 예측 정확도(0.5% MAE) 개선보다
- **상한율 4번째 소수점 정밀도**가 실제 낙찰 결정자

이것이 방안 C (가정 사정률 상한율 모델)이 가장 큰 지렛대인 이유.

## 사정률 정상 범위

- **정상 데이터**: `br1 BETWEEN 95 AND 105` (= 사정률 -5% ~ +5%)
- **이상치**: 5% 이상 차이 = 비정상 입찰 (내역입찰, 시담 등)
- 통계 계산 시 항상 이 필터 적용
